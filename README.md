# HOW TO RUN

## Install dependencies

First, install all node modules

```bash
$ pnpm install
```

Then, set API url in config.json file:

```json
{
  "api": "http://localhost:3001"
}
```

To run in dev mode

```bash
$ pnpm dev
```

To build and run in production mode

```bash
$ pnpm build
$ pnpm serve
```
